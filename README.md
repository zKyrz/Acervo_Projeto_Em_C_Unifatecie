Gerenciador de Biblioteca Pessoal
Este projeto é uma ferramenta de linha de comando desenvolvida em Linguagem C para organizar e gerenciar um inventário pessoal de livros. O sistema permite listar os títulos cadastrados e realizar a remoção de itens (doação/descarte) de forma eficiente.

🚀 Funcionalidades
Listagem de Acervo: Visualização organizada de todos os livros cadastrados com seus respectivos IDs.

Gestão de Inventário: Funcionalidade para remover livros do acervo através de identificação única (ID).

Tratamento de Erros: Proteção contra entradas inválidas e validação de existência de registros.

Código Limpo: Estrutura modular, profissional e devidamente comentada.

🛠️ Tecnologias Utilizadas
Linguagem: C (Padrão ANSI/ISO)

📋 Como utilizar
Ao iniciar o programa, o menu principal será exibido.

Escolha a opção 1 para listar todos os livros atualmente cadastrados.

Escolha a opção 2 para remover um livro. O sistema solicitará o ID do livro; insira o número correspondente ao item que deseja doar.

Escolha a opção 3 para encerrar o programa.
